# 서울 카드 데이터 기반 트렌드 지역 분석 및 개인 맞춤 추천 서비스
## PPT 내용 정리

---

## 1. 프로젝트 개요

### 1.1 프로젝트 배경

#### 문제점
- **"소문이후" 문제**: 트렌드 지역을 사후에 발견하는 문제
  - 이미 유명해진 후에야 알게 됨
  - 선제적 정보 부족으로 기회 상실

- **"데이터부재" 문제**: 객관적 데이터 기반 의사결정 어려움
  - 주관적 판단에 의존
  - 정량적 근거 부족

- **"개인화부재" 문제**: 개인 맞춤 추천 부재
  - 일반적인 추천만 제공
  - 사용자 특성 반영 미흡

#### 해결 방안
- 서울시 카드 소비 데이터 + 실시간 상권현황데이터 통합 분석
- 데이터 기반 객관적 트렌드 지역 탐지
- 개인 맞춤 추천 시스템 구축

### 1.2 데이터 구조 및 활용 방식

#### 데이터 소스 1: 서울특별시 빅데이터 캠퍼스
- **데이터 유형**: 서울시민의 업종별 카드소비 데이터
- **URL**: https://buly.kr/6tcj5i5
- **수집 기간**: 2024.01 ~ 2025.06 (547일)
- **주요 컬럼**:
  - 기준일자, 가맹점주소시군구, 업종대분류
  - 카드이용금액계, 카드이용건수계
  - 성별, 연령대, 시간대 등

#### 데이터 소스 2: 서울 열린데이터광장
- **데이터 유형**: 서울시 실시간 상권현황데이터
- **API URL**: https://data.seoul.go.kr/dataList/OA-22385/F/1/datasetView.do
- **주요 컬럼**:
  - 핫스팟 정보: AREA_NM (장소명), AREA_CD (코드)
  - 실시간 상권 현황: AREA_CMRCL_LVL, AREA_SH_PAYMENT_CNT
  - 업종 정보: RSB_LRG_CTGR (대분류), RSB_MID_CTGR (중분류)
  - 인구통계: CMRCL_MALE_RATE, CMRCL_FEMALE_RATE
  - 연령대별 비율: CMRCL_10_RATE ~ CMRCL_60_RATE
  - 업데이트 시간: CMRCL_TIME

#### 활용 방식
- **카드 데이터**: 과거 패턴 분석, 트렌드 탐지, 집단 프로필 생성
- **실시간 상권데이터**: 현재 상태 파악, 실시간 지역 프로필 생성

## 2. 정적 집단 프로필 생성

### 2.1 카드 데이터 활용

#### 데이터 전처리
- 547일간의 일별 소비 데이터 로드
- 서울시 25개 구 필터링
- 업종별, 지역별, 시간대별 집계

#### 주요 분석 지표
1. **지역별 특성 분석**
   - 특화 업종 식별 (특화 비율 30% 이상)
   - 업종 다양성 계산 (고유 업종 수)
   - 소비 안정성 측정 (변동계수)

2. **성장률 분석**
   - 2024 상반기 vs 하반기 비교
   - 지역별 성장률 계산
   - 상승 지역 식별 (성장률 > 1.5%)

3. **변동계수 분석**
   - 일별 소비 금액의 변동성 측정
   - 안정성 구분: 매우안정(<15%), 안정(15-20%), 보통(20-30%), 변동성높음(≥30%)

### 2.2 집단 소비 성향 추출

#### 연령대별 페르소나 (7개 그룹)
1. **30-39세**: 최대 소비층
   - 주요 업종: 한식(30.1%), 보험(14.8%), 안마/마사지(10.6%)
   - 특징: 다양한 업종 소비, 높은 소비액

2. **40-49세**: 가정 중심 소비
   - 주요 업종: 대형마트(44.2%), 한식(11.4%), 주유소(6.6%)
   - 특징: 가정용품 구매 중심

3. **50-59세**: 식품 중심
   - 주요 업종: 기타식품(36.2%), 한식(24.8%), 양식(7.7%)
   - 특징: 식품 구매 비중 높음

4. **20-29세**: 젊은 층
   - 주요 업종: ZZ_나머지(34.4%), 한식(25.0%), 기타요식(12.6%)
   - 특징: 다양한 경험 추구

5. **60-69세**: 실용적 소비
   - 주요 업종: LPG가스(29.3%), 한식(18.4%), 슈퍼마켓일반형(14.3%)
   - 특징: 필수품 중심

6. **70세이상**: 기본 소비
   - 주요 업종: 슈퍼마켓일반형(27.9%), 기타요식(24.9%)
   - 특징: 최소한의 소비

7. **20세미만**: 외식 중심
   - 주요 업종: 한식(29.7%), 편의점(26.1%), 일식(21.2%)
   - 특징: 간편식 선호

#### 성별 페르소나
- **남성**: 한식(25.0%), 대형마트(17.1%), 기타식품(12.8%)
- **여성**: 한식(14.8%), ZZ_나머지(11.9%), 컴퓨터/소프트웨어(11.4%)

#### 시간대별 패턴
- **점심(12-18시)**: ZZ_나머지, 결제대행(PG), 전자상거래
- **오후(18-24시)**: 전자상거래, 주유소, 한식
- **저녁**: 전자상거래, 한식, 백화점

#### 요일별 패턴
- **주말 소비**: 평일 대비 34.4% 증가
- **주말 추천**: 외식, 쇼핑, 여가 활동 업종 우선

### 📈 PPT 2장: 지역 점수 계산 알고리즘 및 실제 계산 예시

#### 3.3 지역 활성도 점수 계산 (상세)

**1. 상권 활성도 점수 (Commercial Activity Score)**

**계산 공식:**
```
상권활성도 = 정규화(결제건수) × 0.4 + 정규화(결제금액) × 0.3 + 정규화(업종다양성) × 0.3
```

**정규화 방법:**
- 각 지표를 0~100 범위로 정규화
- 최대값 기준 상대 점수 계산

**실제 계산 예시 (광화문·덕수궁):**

| 지표 | 실제 값 | 정규화 값 | 가중치 | 가중 점수 |
|------|---------|----------|--------|----------|
| 결제건수 | 92건 | 65.0점 | 0.4 | 26.0점 |
| 결제금액 | 3,450,000원 | 70.0점 | 0.3 | 21.0점 |
| 업종다양성 | 8개 업종 | 80.0점 | 0.3 | 24.0점 |
| **상권활성도 점수** | - | - | - | **71.0점** |

**계산 상세:**
- 결제건수: 92건 → 전체 핫스팟 평균 대비 65% → 65점
- 결제금액: 3,450,000원 → 전체 핫스팟 평균 대비 70% → 70점
- 업종다양성: 8개 업종 → 최대 10개 기준 80% → 80점

**2. 업종별 특화 점수 (Industry Specialization Score)**

**계산 공식:**
```
특화점수 = (해당업종결제건수 / 전체결제건수) × 100
```

**실제 계산 예시 (광화문·덕수궁):**

| 업종 | 결제 건수 | 전체 대비 비율 | 특화 점수 | 특화 등급 |
|------|----------|--------------|----------|----------|
| 여가·오락 (스포츠/문화/레저) | 36건 | 39.1% | 39.1점 | 중간 트렌드 |
| 한식 | 12건 | 13.0% | 13.0점 | 일반 지역 |
| 편의점 | 13건 | 14.1% | 14.1점 | 일반 지역 |
| 제과/커피/패스트푸드 | 8건 | 8.7% | 8.7점 | 일반 지역 |
| 할인점/슈퍼마켓 | 8건 | 8.7% | 8.7점 | 일반 지역 |
| 기타요식 | 7건 | 7.6% | 7.6점 | 일반 지역 |
| 일식/중식/양식 | 4건 | 4.3% | 4.3점 | 일반 지역 |

**특화 등급 기준:**
- **강한 트렌드 지역**: 특화 비율 50% 이상
- **중간 트렌드 지역**: 특화 비율 30-50%
- **일반 지역**: 특화 비율 30% 미만

**3. 인구통계 점수 (Demographic Score)**

**계산 공식:**
```
인구통계점수 = Σ(연령대별비율 × 가중치) × 100
```

**가중치 체계:**
- **20-30대**: 가중치 1.5 (활발한 소비층, 높은 소비력)
- **40-50대**: 가중치 1.2 (안정적 소비층, 안정적 소비)
- **10대 이하, 60대 이상**: 가중치 1.0 (기본 가중치)

**실제 계산 예시 (광화문·덕수궁):**

| 연령대 | 소비 비율 | 가중치 | 가중 점수 |
|--------|----------|--------|----------|
| 10대 이하 | 0.3% | 1.0 | 0.3점 |
| 20대 | 15.9% | 1.5 | 23.9점 |
| 30대 | 32.0% | 1.5 | 48.0점 |
| 40대 | 31.9% | 1.2 | 38.3점 |
| 50대 | 8.5% | 1.2 | 10.2점 |
| 60대 이상 | 11.4% | 1.0 | 11.4점 |
| **인구통계 점수** | - | - | **132.1점** |

**해석:**
- 20-30대 비율이 높아 활발한 소비층 집중 (47.9%)
- 40대 비율도 높아 안정적 소비층 포함 (31.9%)
- 종합적으로 활발한 소비 지역으로 평가

**4. 종합 지역 점수 (Comprehensive Area Score)**

**계산 공식:**
```
종합점수 = (상권활성도 × 0.4) + (특화점수 × 0.3) + (인구통계점수 × 0.3)
```

**실제 계산 예시 (광화문·덕수궁):**

| 지표 | 점수 | 가중치 | 가중 점수 |
|------|------|--------|----------|
| 상권활성도 | 71.0점 | 0.4 | 28.4점 |
| 특화점수 (최고 업종) | 39.1점 | 0.3 | 11.7점 |
| 인구통계점수 | 132.1점 | 0.3 | 39.6점 |
| **종합 지역 점수** | - | - | **79.7점** |

**점수 해석:**
- 80점 이상: 매우 활성화된 지역 (Hot Zone)
- 60-80점: 활성화된 지역 (Active Zone)
- 40-60점: 보통 지역 (Normal Zone)

## 4. 사용자 기반 입력 매칭

### 4.1 집단 패턴과의 매칭

#### 사용자 입력 정보
1. **인구통계 정보**
   - 연령대 (10대미만, 10대, 20대, 30대, 40대, 50대, 60대이상)
   - 성별 (남성, 여성)

2. **선호 정보**
   - 선호 업종 (1-3개)
   - 선호 시간대 (선택)
   - 선호 요일 (평일/주말)

3. **현재 상황**
   - 현재 시간
   - 현재 위치 (선택)

#### 매칭 알고리즘

**1. 페르소나 매칭**
```
페르소나점수 = 연령대매칭(40%) + 성별매칭(30%) + 업종선호도(30%)
```
- 연령대별 페르소나와의 유사도 계산
- 성별 페르소나와의 유사도 계산
- 선호 업종과 페르소나 주요 업종의 일치도

**2. 시간대 매칭**
```
시간대점수 = 시간대적합도 × 시간대별소비패턴
```
- 현재 시간대의 업종별 소비 패턴 반영
- 주말/평일 패턴 반영

**3. 지역 특성 매칭**
```
지역매칭점수 = 업종특화도(50%) + 인구통계일치도(30%) + 활성도(20%)
```
- 사용자 선호 업종과 지역 특화 업종의 일치도
- 사용자 연령대와 지역 주요 연령대의 일치도
- 지역 활성도 (높을수록 추천 우선순위 상승)

**4. 종합 매칭 점수**
```
종합매칭점수 = (페르소나점수 × 0.4) + (시간대점수 × 0.3) + (지역매칭점수 × 0.3)
```

## 5. 개인화 추천

### 5.1 데이터 결합 및 추천 생성

#### 데이터 통합
1. **카드 데이터 + 실시간 상권데이터**
   - 카드 데이터: 과거 패턴, 트렌드, 안정성
   - 실시간 데이터: 현재 상태, 활성도
   - 통합 분석을 통한 종합 평가

2. **집단 프로필 + 지역 프로필**
   - 사용자 집단 특성과 지역 특성 결합
   - 개인 맞춤 지역 추천 생성

#### 추천 알고리즘

**1. Hot Zone Dashboard (트렌드 지역 탐지)**
```
트렌드지역점수 = (특화비율 × 0.4) + (업종다양성 × 0.3) + (성장률 × 0.3)
```
- 특화 비율 50% 이상 = Hot Zone
- 업종 다양성 높음 = 종합 Hot Zone
- 실시간 활성도 높음 = 활성 Hot Zone

**2. Rising Zone Forecast (상승 지역 예측)**
```
상승지역점수 = (현재성장률 × 0.4) + (예측성장률 × 0.6)
예측성장률 = (안정성가중치 × 성장률) + (특화업종성장률 × 0.2)
```
- 성장률 > 1.5% + 안정성 높음(변동계수 < 20%) = 상승 지역
- 변동계수 < 20% = 높은 신뢰도

**3. Personal Pick (개인 맞춤 추천)**
```
개인화점수 = (페르소나매칭 × 0.4) + (시간대적합도 × 0.3) + (지역특화도 × 0.3)
최종추천점수 = (개인화점수 × 0.6) + (상승지역점수 × 0.4)
```
- 사용자 프로필 기반 맞춤 추천
- 시간대, 요일 패턴 반영
- 상승 지역 우선 추천

#### 추천 결과 구성

**1. 추천 지역 리스트**
- 상위 10개 지역 추천
- 각 지역별 추천 이유 표시
- 지역별 상세 정보 제공

**2. 추천 업종 리스트**
- 지역별 특화 업종 추천
- 사용자 선호 업종과의 매칭도 표시

**3. 예상 소비 패턴**
- 시간대별 예상 소비 금액
- 요일별 예상 소비 패턴
- 연령대별 유사 사용자 소비 패턴
