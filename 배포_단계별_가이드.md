## 📝 배포 단계

### 1️⃣ Vercel 웹사이트 접속

**URL**: https://vercel.com

1. **Sign Up** 또는 **Log In** 클릭
2. **GitHub 계정으로 로그인** 선택 (권장)
3. Vercel에 GitHub 접근 권한 허용

---

### 2️⃣ 프로젝트 Import

1. **Dashboard**에서 **Add New...** 버튼 클릭
2. **Project** 선택
3. **Import Git Repository** 탭 선택

---

### 3️⃣ GitHub 저장소 선택

1. 저장소 목록에서 **`SUE960/space_recommendation`** 찾기
   - 보이지 않으면: **Adjust GitHub App Permissions** 클릭 후 권한 부여
2. **Import** 버튼 클릭

---

### 4️⃣ 프로젝트 설정

다음 설정이 자동으로 감지됩니다:

```
Framework Preset: Next.js
Root Directory: ./
Build Command: npm run build
Output Directory: .next
Install Command: npm install
```

**추가 설정 (선택사항)**:
- **Project Name**: space-recommendation (또는 원하는 이름)
- **Environment Variables**: 필요 시 추가

---

### 5️⃣ 배포 시작

1. **Deploy** 버튼 클릭
2. 빌드 진행 상황 확인 (약 2-3분 소요)
3. 배포 완료 대기

---

### 6️⃣ 배포 완료 확인

배포가 완료되면:
- ✅ **배포 URL** 자동 생성 (예: `https://space-recommendation-xxx.vercel.app`)
- ✅ **Visit** 버튼으로 사이트 확인
