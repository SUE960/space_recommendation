# 부록 D: 주요 발견 사항

## 💡 **예상 밖의 인사이트 TOP 5**

### **1. 관악구 = 20대 천국**
```
실제 데이터:
- 20대: 36.8%
- 30대: 39.6%
→ 총 76.4%가 2030세대!

→ 서울대, 숭실대 등 대학가 특성 명확히 반영
```

### **2. 마포구 ≠ 홍대 (20대 중심 아님!)**
```
기존 인식: 홍대 = 20대 중심
실제 데이터:
- 40대: 42.8% ← 가장 많음!
- 여성: 82.9%

→ 실제로는 40대 여성이 주요 소비층
→ "홍대"는 일부 지역일 뿐, 마포구 전체는 다름
```

### **3. 성동구 = 30대 신혼부부 밀집 지역**
```
실제 데이터:
- 30대: 87.0% ← 서울 최고!

→ 성수동 등 신혼부부 선호 지역
→ 예상 외로 30대 집중도 최고
```

### **4. 강남구 순위 하락 이유**
```
Level 1 (품질): 7위
20대 추천: 5위 (소득 불일치)
40대 추천: 8위 (매칭 낮음)

→ 특화도 낮음 (여러 업종 골고루)
→ 고소득자에게만 적합
→ "개인화"의 중요성 입증
```

### **5. B급 vs C급의 차이**
```
B급 6개 구: 명확한 강점 보유
- 중구: 다양성
- 마포구: 성장률
- 영등포구: 직장인
- 서초구: 특화도
- 관악구: 젊은층
- 동작구: 생활밀착

C급 15개 구: 무난하지만 특징 없음
→ 차별화 전략 필요
```
