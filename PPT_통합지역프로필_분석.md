# 슬라이드 1: 분석 개요

## 통합 실시간 지역 프로필 생성 시스템

### 📊 **최종 목표**
**개인화 추천 시스템 구축**
- 사용자 특성 기반 **최적 지역 매칭**
- 집단 패턴을 활용한 **개인화 추천**
- "나에게 맞는 지역은?" 질문에 답하기

### 🔗 **통합 데이터 (4가지)**
1. **카드 소비 데이터** - 실제 경제 활동 패턴
2. **상주인구 데이터** - 인구 구조 및 밀도
3. **소득·소비 데이터** - 경제력 및 소비 성향
4. **GIS 영역 데이터** - 지리적 공간 정보

### 🎯 **2단계 분석 구조**

**Level 1: 지역 객관적 평가**
- 질문: "이 지역이 얼마나 좋은 상권인가?"
- 용도: 투자/창업 의사결정, 상권 분석

**Level 2: 사용자-지역 매칭** ⭐ **핵심**
- 질문: "이 사용자에게 얼마나 적합한가?"
- 용도: 개인화 추천, 맞춤형 서비스

**최종 추천 = Level 1 × Level 2**

# 슬라이드 2-1: Level 1 - 지역 객관적 평가

## 지역 품질 점수 (Regional Quality Score)

### 🎯 **목적**
- 지역 자체의 **객관적 매력도** 평가
- 상권의 **인프라, 활성도, 경제력** 측정
- **모든 사용자에게 동일한 점수**

### 💡 **핵심 개념**
```
점수가 높다 = 좋은 상권, 발전된 지역
하지만 모든 사용자에게 적합한 것은 아님!
```

**예시:**
- 강남역 91.5점 = 매우 우수한 상권
- 하지만 20대 학생에게는 홍대가 더 적합할 수 있음

---

## 4대 평가 지표 (0-100점)

### 📈 **1. 상업활동 점수 (가중치 30%)**

**산출 공식:**
```
상업활동 점수 = (결제건수_정규화 × 0.4) + (결제금액_정규화 × 0.4) + (활성업종수_정규화 × 0.2)
```

**세부 계산:**
- **결제건수 정규화**: `(해당지역_결제건수 / 전체_최대값) × 100`
- **결제금액 정규화**: `(해당지역_결제금액 / 전체_최대값) × 100`
- **활성업종수 정규화**: `(해당지역_업종수 / 전체_최대값) × 100`

**의미:**
- 상권의 **거래 활성도**를 나타냄
- 높을수록 **활발한 경제 활동** 발생
- 객관적 데이터 (카드 거래 내역)

# 슬라이드 2-2: 분석 로직 - 점수 산출 체계

### 🎯 **2. 특화도 점수 (가중치 25%)**

**산출 공식:**
```
특화도 점수 = (최대_업종_비율 × 0.6) + (상위3업종_집중도 × 0.4)
```

**세부 계산:**
- **최대 업종 비율**: `(1위_업종_결제액 / 전체_결제액) × 100`
- **상위3업종 집중도**: `(상위3업종_결제액_합 / 전체_결제액) × 100`

**의미:**
- 상권의 **업종 특화 정도**를 측정
- 높을수록 **명확한 업종 정체성** 보유

**예시:**
- 명동: 의류·패션 특화 (특화도 85.6점)
- 강남역: 금융·비즈니스 특화 (특화도 88.7점)

# 슬라이드 2-3: 분석 로직 - 점수 산출 체계

### 👥 **3. 인구통계 점수 (가중치 20%)**

**산출 공식:**
```
인구통계 점수 = (인구밀도_점수 × 0.5) + (주요연령대_비율 × 0.3) + (가구특성_점수 × 0.2)
```

**세부 계산:**
- **인구밀도 점수**: `min((지역인구밀도 / 서울평균밀도) × 50, 50)`
- **주요연령대 비율**: `(최대_연령대_인구 / 전체인구) × 100`
- **가구특성 점수**: `(1인가구_비율 × 경제활동_가중치)`

**활용 데이터:**
- 총인구: 9,360,421명
- 인구밀도: 15,452명/㎢
- 연령대별 분포: 10대~60대이상 6개 그룹
- 가구당 인구: 2.1명

# 슬라이드 2-4: 분석 로직 - 점수 산출 체계 ⭐ 신규

### 💰 **4. 경제력 점수 (가중치 25%) - 신규 추가**

**산출 공식:**
```
경제력 점수 = (소득수준_점수 × 0.5) + (소비활성도_점수 × 0.5)
```

**세부 계산:**
- **소득수준 점수**: `min((해당지역_월평균소득 / 전국평균소득) × 50, 50)`
  - 전국 평균 소득: 3,384,950원
  
- **소비활성도 점수**: `min((지역_지출총액 / 10억원) × 50, 50)`
  - 기준: 10억원 이상 = 만점(50점)

**의미:**
- 지역 주민의 **구매력**을 평가
- 높을수록 **고소득·고소비** 상권

**실제 데이터 예시:**
- 서울국제고등학교: 월평균소득 2,838,790원, 지출 18.9억원
- 성균관대학교: 월평균소득 3,094,492원, 지출 10.6억원

# 슬라이드 2-5: Level 1 - 지역 품질 종합 점수

### 🏆 **지역 품질 종합 점수 (Regional Quality Score)**

**최종 산출 공식:**
```
지역품질점수 = (상업활동 × 0.30) + (특화도 × 0.25) + (인구통계 × 0.20) + (경제력 × 0.25)
```

**가중치 설계 근거:**

| 점수 | 가중치 | 이유 |
|------|--------|------|
| 상업활동 | 30% | 실제 거래 데이터로 **현재 활성도** 측정 |
| 경제력 | 25% | **구매력**과 소비 잠재력 평가 |
| 특화도 | 25% | 상권의 **차별화** 요소 |
| 인구통계 | 20% | **타겟 고객** 규모 파악 |

**점수 해석:**
- **90점 이상**: 초우량 상권 (인프라 최상급)
- **80-89점**: 우량 상권 (안정적 환경)
- **70-79점**: 보통 상권 (기본 인프라)
- **70점 미만**: 발전 필요 지역

### ⚠️ **중요: 이것은 Level 1일 뿐!**
```
강남역 91.5점 ≠ 모든 사람에게 최고
→ Level 2 (사용자 매칭)이 필요!
```

## AI 기반 타겟팅 시스템

**3단계 매칭 프로세스:**

#### **Step 1: 소비 패턴 기반 타겟팅**
```python
if 식료품 > 20%:

    타겟 = "학생/학부모층"
...
```

#### **Step 2: 경제력 기반 세분화**
```python
if 경제력점수 >= 70:
    계층 = "고소득층"
elif 경제력점수 >= 50:
    계층 = "중상위소득층"
else:
    계층 = "중하위소득층"
```

```python
주요_연령대 = max(10대, 20대, ..., 60대이상)
타겟 += f"{주요_연령대} 중심"
```

# 슬라이드 5: 핵심 인사이트 및 시사점

## 분석을 통해 발견한 것들

### 🔍 **1. 경제력의 중요성**
- 모든 주요 상권이 **경제력 97.9점**으로 최고 수준
- **구매력이 높은 지역**일수록 다양한 업종 진입 가능
- 소득·소비 데이터 통합으로 **숨겨진 잠재력 발굴**

### 📊 **2. 상권별 차별화 전략 필요**
- 종합점수는 비슷해도 **특화 영역은 상이**
- 강남역(비즈니스) vs 홍대(문화) vs 명동(관광)
- **타겟 맞춤형 접근**이 성공의 열쇠

### 💡 **3. 데이터 통합의 효과**
**단일 데이터만 사용할 때:**
- 카드 데이터만 → 거래량만 파악
- 인구 데이터만 → 잠재 고객 규모만 추정

**4개 데이터 통합 시:**
- ✅ **누가** (인구통계)
- ✅ **얼마나** (경제력)
- ✅ **무엇을** (소비패턴)
- ✅ **어디서** (GIS + 카드데이터)

→ **입체적 상권 분석** 가능!
