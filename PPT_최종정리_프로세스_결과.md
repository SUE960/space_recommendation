# 슬라이드 1: 전체 프로세스 개요

## 📋 **4단계 개인화 추천 프로세스**

```
INPUT: 사용자 프로필 (나이, 소득, 취향)
   ↓
STEP 1: 데이터 수집 및 통합
   ↓
STEP 2: Level 1 - 지역 객관적 평가
   ↓
STEP 3: Level 2 - 사용자-지역 매칭
   ↓
STEP 4: 최종 추천 점수 계산
   ↓
OUTPUT: 맞춤형 지역 TOP 3 + 추천 이유
```

# 슬라이드 3: STEP 2 - Level 1 지역 품질 평가

## 📊 **4개 지표로 지역 객관적 평가**

### **평가 공식**

```
종합점수 = (상업활동 × 30%) + (특화도 × 25%) + 
          (인구통계 × 20%) + (경제력 × 25%)
```

### **지표별 설명**

| 지표 | 비중 | 측정 내용 | 데이터 출처 |
|------|------|-----------|------------|
| **상업활동** | 30% | 업종 다양성 + 소비 안정성 | 파일 1 + CV |
| **특화도** | 25% | 주요 업종 집중도 | 기존 분석 |
| **인구통계** | 20% | 인구 활동 안정성 | CV + 인구 API |
| **경제력** | 25% | 성장률 + 발전 가능성 | 성장률 + 소득 API |

### **결과: 25개 구 품질 순위**

| 순위 | 구 | 종합점수 | 등급 | 특징 |
|:----:|:---|:--------:|:----:|:-----|
| 1 | 중구 | 66.9 | B급 | 업종 다양성 최고 (15개) |
| 2 | 마포구 | 64.8 | B급 | 문화 특화, 성장률 +1.67% |
| 3 | 영등포구 | 64.1 | B급 | 직장인 밀집, 성장률 +2.01% |
| 7 | **강남구** | 59.4 | C급 | 학원 특화, 다양성 낮음 |

**💡 인사이트:**
- 강남구가 7위? → 특화도 낮음 (여러 업종 골고루)
- 이것이 Level 1의 한계 → Level 2에서 개인화 필요!

# 슬라이드 4: STEP 3 - Level 2 사용자 매칭

## 🎯 **4개 지표로 사용자-지역 적합도 평가**

### **매칭 공식**

```
매칭점수 = (인구통계 × 40%) + (소비패턴 × 35%) + 
          (소득수준 × 15%) + (업종선호 × 10%)
```

### **지표별 매칭 로직**

#### **1. 인구통계 매칭 (40%)** - 가장 중요!

```python
# 같은 연령대가 많을수록 높은 점수
사용자_연령대 = "20대"
지역_20대_비율 = 36.8%  # 실제 카드 사용 데이터

점수 = min(36.8 × 2, 100) = 73.6점
```

**데이터:** 파일 6 (성별 연령대별) → 100,101건 카드 거래 분석

---

#### **2. 소비패턴 매칭 (35%)** - 코사인 유사도

```python
# 벡터 유사도 계산
사용자_소비 = [식료품:33, 교통:18, 생활용품:22, 기타:27]
지역_소비 = [식료품:33, 교통:18, 생활용품:22, 기타:27]

코사인_유사도 = 0.99 → 99점!
```

**데이터:** 파일 1 (소비 지역별) → 155,895건 소비 데이터 분석

---

#### **3. 소득수준 매칭 (15%)** - 가격대 적정성

```python
비율 = 사용자_소득 / 지역_평균소득

if 0.8 ≤ 비율 ≤ 1.2:  # ±20% 이내
    점수 = 100점  # 적정!
```

**데이터:** 소득·소비 API (42,346개 상권)

---

#### **4. 업종선호 매칭 (10%)** - 교집합 비율

```python
사용자_선호 = {카페, 술집, 클럽}
지역_특화 = {카페, 술집, 공연장}

교집합 = {카페, 술집} → 2/3 = 67점
```

# 슬라이드 5: STEP 4 - 최종 추천 계산

## 🏆 **Level 1 × Level 2 = 최종 점수**

### **공식**

```
최종_추천_점수 = (Level 1_품질 / 100) × Level 2_매칭
```

### **계산 예시: 20대 학생 → 관악구**

```
Level 1 (지역 품질): 63.0점
├─ 상업활동: 37.4
├─ 특화도: 90.5 (일식 특화)
├─ 인구통계: 45.6
└─ 경제력: 80.0 (성장률 +2.00%)

Level 2 (매칭도): 72.1점
├─ 인구통계: 75 (20대 36.8%, 30대 39.6% - 대학가!)
├─ 소비패턴: 90 (실제 소비 데이터 일치)
├─ 소득수준: 70 (가격대 적정)
└─ 업종선호: 0

최종점수 = (63.0 / 100) × 72.1 = 45.4점
```

**결과: 1위 추천!** 🥇
