## 🚀 배포 방법

### Option 1: Vercel (권장)

#### 프론트엔드
```bash
# 1. GitHub에 푸시
git push origin main

# 2. Vercel 대시보드
# - https://vercel.com 접속
# - "New Project" 클릭
# - GitHub 저장소 연결
# - 자동 배포 완료

# 3. 환경 변수 설정
# Settings → Environment Variables
# NEXT_PUBLIC_API_URL=https://your-api-server.com
```

#### 백엔드 (Railway)
```bash
# 1. Railway CLI 설치
npm install -g @railway/cli

# 2. 로그인 및 배포
railway login
cd api
railway init
railway up

# 3. 환경 변수 설정
# Railway 대시보드에서 설정
```

---

### Option 2: 로컬 테스트

```bash
# 터미널 1: 백엔드
cd api
uvicorn main:app --reload --port 8000

# 터미널 2: 프론트엔드
npm run dev

# 브라우저
http://localhost:3000
```
